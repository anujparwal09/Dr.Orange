## рҹӨҜ What Even IS This?

Imagine a doctor. Now imagine that doctor went to med school for oranges specifically. Now imagine that doctor never sleeps, never misses a diagnosis, runs on GPU, and costs less than a chai.

**That's Dr. Orange.**

Point your camera at any orange. In **< 3 seconds**, our Multi-Task Learning CNN вҖ” built on MobileNetV2 вҖ” simultaneously fires **4 AI prediction heads** to tell you:

- рҹҰ  **What disease does it have?** (with 97.7% accuracy)
- рҹ’Ҝ **What's its quality score?** (1вҖ“10 commercial grade)
- рҹ“… **How many days before it rots?** (shelf life forecast)
- рҹҺҜ **How ripe is it?** (Unripe вҶ’ Near-Ripe вҶ’ Ripe вҶ’ Overripe)

The CNN prediction results are then passed to **Gemini 2.5 Flash**, which generates a full clinical report, treatment protocol, and prevention guide вҖ” in plain English AND Hindi.

## рҹҡҖ Getting Started (Zero to Diagnosis in 5 Minutes)

### Prerequisites
```
Python 3.11+ | Node.js 18+ | Git | A love for oranges рҹҚҠ
```

### 1пёҸвғЈ Clone the Repo
```bash
git clone https://github.com/anujparwal09/Dr.Orange.git
cd Dr.Orange
```

### 2пёҸвғЈ Backend Setup
```bash
cd backend

# Create & activate virtual environment
python -m venv venv
source venv/bin/activate       # Linux/Mac
.\venv\Scripts\activate        # Windows

# Install all dependencies
pip install -r requirements.txt

# Setup environment variables
cp .env.example .env
# Now edit .env with your keys (see below)

# Fire it up рҹ”Ҙ
python app.py
```

### 3пёҸвғЈ Frontend Setup
```bash
cd frontend

# Install dependencies
npm install

# Create env file
echo "NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_client_id" > .env.local

# Launch dev server
npm run dev
```

### 4пёҸвғЈ Download the ML Model
Download `orange_mtl_model.keras` and drop it into `backend/model/`.

> **Without the model, the backend will boot but predictions won't work.**
> Check `GET /health` вҖ” `"model_loaded": true` means you're good to go. вң…
